Abstract: Firefox Developer Edition es una versión de Firefox que agrega nuevas y experimentales carecterísticas que simplifican en proceso de desarrollo Web, ya sea de aplicaciones Web para escritorio como aplicaciones web móviles. En este taller se mostrarán a los nuevos Desarrolladores Web cómo usar las herramientas de desarrollo que se encuentran actualmente en Firefox Developer Edition y a los desarrolladores experimentados cómo mejorar el proceso de desarrolo de sus páginas o applicaciones Web con las DevTools de Firefox.
